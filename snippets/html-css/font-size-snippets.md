# A font-size snippet

## Using rem unit to set font-size

### CSS

```css
/*.sm for small; 
.md for medium; 
.lg for large; 
.xl for extra large.*/
.sm {
  font-size: 1rem;
}

.md {
  font-size: 2rem;
}

.lg {
  font-size: 3rem;
}

.xl {
  font-size: 4rem;
}
```

### HTML

`<h2 class="md">Supported Languages</h2>`
